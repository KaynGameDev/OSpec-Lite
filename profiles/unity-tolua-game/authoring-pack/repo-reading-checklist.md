# 读库检查清单

## 顶层结构
- 识别顶层目录中哪些是真正主区，哪些是临时区、辅助区或一次性工具区。
- 判断哪些目录应明确排除出核心项目文档范围。

## 入口与流程
- 从 `Script/MJGame.lua` 出发确认主入口。
- 找到真实主流程入口、UI 切换入口、退出入口。
- 判断主流程是否包含登录、进房、加载、战斗、回宿主等阶段。

## 资源与网络
- 找到资源加载入口、配置映射入口、AssetBundle 或等价资源层入口。
- 找到网络请求、回包、推送的发起与分发入口。

## 生命周期
- 找到 Init / Start / Enter / Exit / Release 等关键生命周期钩子。
- 判断事件、定时器、Action、资源释放的收口位置。

## 边界与技术栈
- 识别宿主平台接口、大厅桥接、第三方系统边界。
- 检查项目是否存在 Unity / ToLua / protobuf / AssetBundle / 桥接层 / 插件式框架等明显特征。
